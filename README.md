# 个人应用展示

这是我的个人应用集合，展示了我在前端开发方面的技能和项目经验。这个项目使用 Next.js 15 构建，包含了多个功能完整的应用界面。

## 🚀 技术栈

- **框架**: Next.js 15 (App Router)
- **UI 组件库**: Ant Design 5
- **样式**: CSS Modules + 全局样式
- **图标**: Ant Design Icons
- **语言**: JavaScript

## 📱 应用展示

### 1. 📚 WeRead App - 读书应用
- **功能**: 微信读书应用界面克隆
- **特色**: 
  - 书籍展示和分类
  - 阅读统计和排行榜
  - 推荐系统
  - 响应式设计
- **技术亮点**: 
  - 使用 Server Components 优化性能
  - 动态路由实现书籍详情页
  - API 路由处理数据请求

### 2. 💬 WeChat App - 聊天应用
- **功能**: 微信聊天界面克隆
- **特色**:
  - 实时聊天界面
  - 消息列表和对话窗口
  - 用户头像和状态显示
  - 聊天侧边栏导航
- **技术亮点**:
  - 客户端组件处理交互
  - CSS Modules 实现样式隔离
  - 组件化架构设计

## 📁 项目结构

```
src/
├── app/
│   ├── api/                    # API 路由
│   │   ├── categories/         # 分类相关 API
│   │   ├── chat/              # 聊天相关 API
│   │   ├── ranking/           # 排行榜 API
│   │   └── recommend/         # 推荐 API
│   ├── chat/                  # 聊天应用
│   │   ├── components/        # 聊天组件
│   │   ├── page.js           # 聊天页面
│   │   └── utils/            # 聊天工具函数
│   ├── reading/              # 读书应用
│   │   ├── components/       # 读书组件
│   │   └── page.js          # 读书页面
│   ├── components/           # 共享组件
│   ├── service/             # 服务层
│   ├── utils/               # 工具函数
│   ├── layout.js            # 根布局
│   └── page.js              # 首页
```

## 🚀 运行项目

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)
